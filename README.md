CBETA_TAFxml
============

以「量化語言統計」為目的之簡化版漢文佛典 XML，由 CBETA XML P5 (https://github.com/cbeta-org/xml-p5.git) 經程式轉出。

專案主持人：洪振洲  
轉換程式撰寫：周邦信  
法鼓佛教學院

| &#x3C;body&#x3E;  |   |   |
| &#x3C;byline&#x3E;  |   |   |
| &#x3C;div&#x3E;  |   |   |
| &#x3C;head&#x3E; |   |   |
| &#x3C;lb&#x3E; |   |   |
